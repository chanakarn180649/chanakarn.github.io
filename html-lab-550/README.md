68102010550 หลังจากที่ผมได้เรียนรู้วิชา CP151 ทำให้โลกของผมได้เปิดกว้างขึ้นถึงการสร้าง Website เพราะผมไม่เคยรู้เรื่องในการสร้าง Website มาก่อน โดยเรื่องนี้เป็น Html ที่เป็นเรื่องพื้นฐานในการเขียน Code สร้าง Webside ในตอนแรกผมคิดว่ามันยากมากๆ เพราะ Function code แต่ละตัวมีการทำงานที่เฉพาะตัว และมันเยอะมาก แต่เมื่อถึงเวลาที่ผมจะต้องทำงานนี้จริงๆ ผมก็สามารถเรียนรู้ Function code ได้ในระดับหนึ่ง แต่ก็ยังจำได้ไม่หมดหรือยังเอาไปใช้ไม่ถูกใน Function code บางตัว (ยกตัวอย่างเช่น Function <input>) ผมไม่รู้ว่าในการเรียนรู้ครั้งต่อไปจะเป็นอย่างไร เพราะผมยังใหม่กับเรื่องนี้ แต่ผมคิดว่าผมสามารถที่จะเรียนรู้ต่อไปได้
